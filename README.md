# Portfolio Blog Website

A personal **Portfolio + Blog Website** built using **Python Django** with **HTML, CSS, and JavaScript** frontend.  
This project showcases profile details, projects, and blog posts with a simple and responsive design.

---

## Tech Stack
- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite  

---

## Features
- Portfolio and About section  
- Blog posting system  
- Projects showcase  
- Contact form  
- Django Admin Panel  
- Responsive UI  

---

## Run Locally
```bash
git clone https://github.com/your-username/PortfolioBlog.git
cd PortfolioBlog
python -m venv venv
venv\Scripts\activate   # Windows
pip install django
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver

Open: http://127.0.0.1:8000/
